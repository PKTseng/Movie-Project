# Movie Shop ( 電影購物網站 )

## 簡介

此作品是 Vue.js 前端練習作品，主要功能以下詳細列出。

前台 ( 目前只做**前台**部分，**後台**還在開發中 )

1. 電影 Banner 有輪播效果
2. 依照使用者選擇電影類型，顯示該類型電影
3. 可將電影加入使用這收藏頁面
4. 點擊該電影可顯示電影詳細資訊頁
5. 購物車
6. 使用者付款區塊

## 使用技術

- Vue CLI
- Vue Router
- VueComponents
- Fetch API
- JavaScript (ES6)
- jQuery
- Bootstrap 4
- RWD
- SCSS
- swiper
- ESLint ( recommended )
- Prettier
